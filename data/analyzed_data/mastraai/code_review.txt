Pattern Type
Multi-agent configuration (code review system)
Deskripsi
Pattern ini mendefinisikan sebuah sistem bernama “Code Review System” yang terdiri dari dua agent: reviewer dan tester. Masing-masing agent memiliki role, instructions, dan model yang digunakan (misalnya gpt-4). Tidak ada definisi eksplisit tentang workflow, task, atau tool; fokus pattern ada pada konfigurasi dan peran agent
