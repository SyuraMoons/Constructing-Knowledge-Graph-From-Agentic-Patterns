Pattern Type
Multi-agent configuration + workflow description 
Deskripsi
Pattern ini mendefinisikan sistem bernama “Customer Support System” yang terdiri dari dua agent (support_agent, escalation_agent) dan sebuah workflow bernama support_flow. Setiap agent memiliki role, instructions, dan model. Workflow hanya didefinisikan dengan name dan description tanpa rincian langkah
