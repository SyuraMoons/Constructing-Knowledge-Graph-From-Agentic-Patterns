Pattern Type
Agent Collaborator Pattern (feedback Context)
Deskripsi
Pattern ini mendefinisikan dua agent (AssistantAgent dan UserProxyAgent) untuk konteks pemberian umpan balik (feedback loop). Agent AI berfungsi sebagai evaluator yang memberi feedback terhadap hasil kerja, sementara UserProxyAgent mewakili pengguna.
