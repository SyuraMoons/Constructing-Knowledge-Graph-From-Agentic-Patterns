Deskripsi
Pattern ini mendefinisikan dua agent: AssistantAgent sebagai AI assistant untuk skenario two_agent_chat, dan UserProxyAgent sebagai representasi user dengan mode input NEVER. Tidak ada task, workflow, atau tool yang diekspresikan di dalam kode; pola ini adalah deklarasi dasar dua agent AutoGen
