Pattern Type
Agent Collaborator Pattern (function Invocation Context)
Deskripsi
Pattern ini mendefinisikan dua agent (AssistantAgent dan UserProxyAgent) untuk skenario pemanggilan fungsi (function calling). Walaupun struktur kodenya sama dengan pattern AutoGen lain, konteks semantiknya berbeda: asisten ini dirancang untuk memanggil fungsi eksternal atau API atas permintaan pengguna.
