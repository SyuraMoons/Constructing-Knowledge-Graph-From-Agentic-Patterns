Pattern Type
Agent Collaboration Pattern (two-agent setup)
Deskripsi
Pattern ini mendefinisikan dua agent: satu AssistantAgent yang berperan sebagai AI assistant, dan satu UserProxyAgent yang mewakili user dengan mode input manusia “NEVER”. Ini adalah struktur AutoGen dasar tanpa workflow, tool, atau task
