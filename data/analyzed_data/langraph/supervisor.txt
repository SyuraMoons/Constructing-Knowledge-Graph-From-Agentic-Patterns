File name
supervisor.py
Pattern Type
Single-node StateGraph (entry = finish)
Deskripsi
mendefinisikan sebuah StateGraph dengan satu node bernama supervisor_node. Node ini menjadi titik masuk (entry point) sekaligus titik akhir (finish point) dari workflow, dengan fungsi pemrosesan identitas (lambda x: x). Tidak ada node lain, edge tambahan, agent, tool, maupun LLM yang dimodelkan eksplisit
