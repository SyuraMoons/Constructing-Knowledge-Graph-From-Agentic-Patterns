Pattern Type
Single-node workflow (analyzer)
Deskripsi
Pattern ini mendefinisikan sebuah StateGraph dengan satu node bernama analyzer_node. Node tersebut berfungsi sebagai entry point sekaligus finish point. Tidak terdapat edge lain atau node tambahan. Struktur ini menggambarkan pola standalone node workflow khas LangGraph
